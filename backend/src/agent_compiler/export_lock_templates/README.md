# Export Lock Templates

This directory stores per-target uv.lock templates generated by the exporter.

- {target}.uv.lock: lockfile reused for exports with matching dependency signature.
- {target}.meta.json: signature metadata used to validate freshness.

Behavior:
- If signature matches, exporter copies template lock into the new export.
- If signature changes (or template missing), exporter runs uv lock once and refreshes template.
